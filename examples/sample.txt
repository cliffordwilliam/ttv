@slide=code
@voice_line=Here is the exact same result, but written in a single line using a list comprehension.
@lang=python
numbers = [1, 2, 3, 4, 5]

squared = [n ** 2 for n in numbers] !#

print(squared)
# Output: [1, 4, 9, 16, 25]
@end
@slide=code
@voice_line=The general syntax is: open bracket, expression, for item in iterable, close bracket. Think of it as writing the for loop inside the list itself.
@lang=python
# Syntax:
# [expression for item in iterable]

# More examples:
doubled = [n * 2 for n in numbers] !#
# [2, 4, 6, 8, 10]

words = ["hello", "world"]
upper = [w.upper() for w in words] !#
# ["HELLO", "WORLD"]
@end

# List Comprehensions — Quick Reference

## Basic syntax
[expression for item in iterable]

## With a condition
[expression for item in iterable if condition]

## Examples
- squares  : [n ** 2 for n in nums]
- evens    : [n for n in nums if n % 2 == 0]
- combined : [n ** 2 for n in nums if n % 2 == 0]

## Rule of thumb
Keep it readable. If it's too long, use a for loop!
